## 开发背景

我是一名已有19年透析经历的尿毒症患者。在长期查阅食物钾、磷含量信息的过程中，我发现网络上的数据往往零散不全，难以满足实际需求。为此，我根据自己的经验整理了可靠的数据，并利用 Vant 与 Vue3 独立开发了这个前端项目，希望能为肾友提供更便捷、准确的查询服务。
